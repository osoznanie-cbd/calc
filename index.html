Что это такое?

Это калькулятор дозировки КБД. Вам нужно указать свой возраст, вес, выбрать недуг из списка, указать дозировку КБД на вашем флакончике и опыт употребления масла. Калькулятор рассчитает рекомендуемую дозировку в зависимости от введённых вами значений :)

Калькулятор дозировки КБД

Пожалуйста, укажите корректный возраст
Пожалуйста, укажите корректный вес
Пожалуйста, выберите хотя бы один недуг
Пожалуйста, выберите дозировку
Пожалуйста, укажите одно из предложенных значений

Рекомендуемая дозировка:

Для вас будет достаточно 0 мг в сутки

Количество капель в сутки - 0

Поместите масло под язык и удерживайте 30-60 секунд, затем проглотите. Главное не касайтесь пипеткой языка, чтобы во внутрь флакончика ничего лишнего не попало :)
